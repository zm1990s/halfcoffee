# 20250216

过年回来，调休导致周末只有一天，忙着 SD-WAN 培训以及越南的邀请，终于告一段落了。

同事说你们部门好忙啊，别的部门都看不到人，你们天天来，有点无奈，我也不想啊。。

不过这两个事忙完后，感觉放松很多，或许是过年出去休息的原因，回来之后人的节奏很慢，开始追剧，刷直播买石头（新爱好），当然还有搞钱（亏钱）。

最终比较上瘾的时候，就只能卸载 App（考虑再三，还是卸载掉了小红书）

搞了个水晶手握，人生第一个直播购物，感觉买的时候有点上头，到了后并不如想象中的那么透，我还是喜欢那种透透的材质吧，透到了心里的那种。回头等这个玩腻了，以及 B 赚钱了在考虑换吧，目标喜马拉雅水晶，如果有茶色的更好。

回来之后，药再用了一周，感觉身上的皮肤病就好了...真的太不容易了。

过年时看某个博主，感觉 A 股并不咋地，让我有些失去信心了（还有个原因是开港户需要存款，至少当时这么觉得），开始第一天卖了一批，卖了后安排越南行程，才发现好像经过 HK，可以走线上办理的途径，就是不知道到时候过关会不会有记录，至少值得尝试一下。

很久没写东西了，没有一点曾经那种焦虑的感觉，躺平真舒服的感觉。看到两个同事在发 Homelab 以及 AI 的内容，稍有些焦虑，不过丝毫没有想着赶上去，难道说工作第二年，年纪到了成老油条了？

# 20250127

度过了忙碌的节前一个月，然后连着躺平三天，实在无聊，刷剧刷到累。状态极其放松。

在上一篇，身体奇痒，后来还是去了医院，诊断为疥疮，一种由疥螨引起的皮肤病，很不好治。医生开的硫磺软膏，使用感受我只能说，相当糟糕。使用第二天开始就有并发症，类似于很久没洗澡，又出了汗的那种痒，忍到第三天洗澡，清爽很多，谁想到后面又感觉有复发，于是开始第二个三天，结果第二个三天依旧没完全好，于是继续...

第三个疗程后，觉得不行了，得换家医院看看，去了华山医院，华山就是专业（但是就医环境很一般，门就那么敞开着，还好这边医生不咋想看我的病灶），说我之前开的口服药副作用大，于是换了一个（确实副作用少，然而止痒效果也一般），外敷的也换了，没成想，我对硫樟软膏过敏（第二天才意识到），原来本来就痒的地方，更加痒了...不过，目测那种奇痒没有了，或许这次根治了？

# 20250104

周末了，上个月连着两周多感觉大家都在积极休假，没多少事，我也趁着这机会继续弄 AI 相关的东西，真的挺上头的。

开源社区还是比较活跃，各种问题和需求，忍不住都想试试他们的需求，于是依次搞了 RAG、数字人、API 工具等一堆东西...虽然挺快乐，但有时候在想，好没有价值...

一直想弄个帮助解读项目代码的 AI，这时候就感觉自己的能力受限了，大致思路知道，但是具体实现就不知道如何开始。

与之类似的，想弄个高仿的数字人，光模型这块看了一堆项目，觉得还是没这么多时间折腾这些。除此之外还有人声替换 TTS，要部署还得用 windows，总感觉有些麻烦。自从用习惯了 API 之后（尤其是免费的），感觉其他都好麻烦。

最近身上各种毛囊炎，每个晚上都是在奇痒中度过，即便如此，还是宁愿自己涂药，懒得去医院...有时候在想，小时候是不是也是同样的问题，冬天异常的难熬？（以前应该手更痒...）

好久没玩游戏里，感觉自己精力就特别好，每天下午基本都不会困，有精力写 PPT，也有精力做实验，整个状态还是不错的。

一个月弄完了 SLR 以及 IOT 的材料，真的累。AI 的东西研究的也算差不多了，够用一会儿了，接下来，可以弄弄重要非紧急的事了。
