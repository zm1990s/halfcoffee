# 20260331

Tech Summit 第二天，第一天如预期的一样充实的度过，上午 Golden Pitch 是一个新加坡人讲平台化，感觉风格非常的引人入胜，感觉他是有做领导的气质，在台上非常自信，他第一名属实实至名归。

下午 Region 的会议，比较意外（也如预期），又被提名一个奖项，感觉像老婆说的，我配得感比较低，所以又是一个双重的感觉，感觉并不是完全好的。

举办 CTF 的时候，本来以为同事只是开场，结果发现直接劫持去讲比赛的内容了，要是讲的好还行，感觉只是蜻蜓点水过了一遍，很多需要注意的细节都没提到，感觉以后共事得小心一些，此人有些危险（有点类似于着急跳墙想突出自己的贡献一样，不过攻击性一般），anyway 不是我喜欢的类型。

晚上吃饭和 K 一桌，本来以为是那种 Formal 的饭局，没想到也是很随意，估计很多人直接 skip 掉出去玩了，本来打算吃完就出去转转，碰到老板，老板说建议多待一会儿，见见人，一开始也不知道见什么，后来拿着酒转了两桌，还真认识一两位新朋友，港台的，也能聊到一些挺好玩的事，比如 HK 的个税真的好严格，咸鱼的交易都可能被收税，突然感觉国内还是挺自由的...

关于 Social 的感觉是，找到共同话题就可以很愉快的聊天，今天的主题就是 AI CTF。

晚饭后去了健身房，碰到了 R，就猜着他会去，9点多的时候健身房几乎没有人了，看来大家来泰国都是来玩的...
健身完毕后和同事去楼下转转，结果碰到了二度去转转的同事，感觉像是要去做一些不正经的事，于是跟随（主要是想出去喝饮料但是没带钱），跟着跟着发现另外两个同事在前面，于是一伙又决定跟随他们，感觉看别人“干坏事”更加刺激。比较可惜的是，跟着跟着发现他们就是去取了个现，然后去马杀鸡...享受不了这种，于是去711买了个冰棍+饮料，冰棍是真难吃，小时候的那种劣质糖味，以及没有搅拌均匀的糖块。东西也不便宜，一个冰棍，一瓶水 RMB 11 多。
回房间稍微休息了下，又无意义的刷了刷短视频，想到白天的事，感觉又有些伤感，更多的应该是慕强和自卑的那种情绪，于是发了个牢骚朋友圈：

> 工作都是容易的，难的是和自己和解[叹气]
> 上帝为你打开一扇窗的同时，会关闭一扇门，就是会让你只能走一条路

即是说，我现在的很多心理问题，就是自己和自己过不去，改变自己是多难的一件事。

晚上睡着没多久开始各种做梦，两度梦到了 N，日有所思夜有所梦，这就是 balance 吧。

工作上，上周是办活动忙碌的一周，不过好在一切都“顺利”结束。

这两周心理咨询，聊到了和家人的关系这部分，觉得有必要再看看巨婴国，再看的时候发现其实很多里面的言论现在并不是很认同，主观的情绪还是比较多，而且会有些强行解读（又或者说生硬的套），感觉或许和10年前第一次看的感觉不太一样。不过还是值得看看，内容相对易懂，而且有些点能让我套在自己或者家人的某些行为之上。

关于家人这部分的收获是，我虽然不喜欢他们的一些思想和行为，但竟然神奇般的继承了某些缺陷下来，比如说话“丧”的情绪很像我妈，还好我身边的人都能容忍我...

另外就是，我会自我愧疚，和之前说的自控力也有一定关系，对于妻子，对于父母，当我想做一些事情的时候会感觉愧疚，其实或许这些事情本身做不做对他们的影响并不大，但在我心里总是会有很多内心戏，这样弄的自己挺有压力，甚至消极。

周一周二去了长沙，那边天气和之前去的时候一样，阴雨，上午剪完客户，下午放松和同事去逛街，五一广场附近打卡，中午吃饭，我发现我竟然是有些害羞的感觉，聊着聊着慢慢就感觉轻松自然了。这可能也是内心戏造成的吧，如果不去想什么，或许就不会有紧张的情绪。一天喝了三杯奶茶，茶颜悦色是很的不错，也便宜，湖南的饭也不错，好吃不贵，中饭竟然只吃了 150 不到。晚上回去前去茶颜悦色买了些小零食，买小零食也是挺快乐的一件事，也不知道为啥以前没有发现这种快乐，或许是因为旁边有个人带着买？

事后再看那天的逛街，感觉自己就像是“谈恋爱”一样，说话做事都会刻意的很谨慎，小心翼翼，生怕说错什么而引发不好的事情。那种纠结，小心，未知的感觉像是回到了从前，想想挺好玩的。

再说回咨询，感觉我就像飞蛾扑火一样，天然会被外向的性格吸引，吸引的同时，会接纳不了自己的封闭（也就是会拿自己和外向的人作比较，发现比不上他们（废话）），然后就会不开心。总之，一切都由心生，我没能接纳自己，一直认为自己需要改变。

算是比较忙的一周？也不知道在忙什么，周内办了比赛，前面看到好多人拿成绩感觉已经自我怀疑了（产品未能防住大家的攻击），后面有一天发现原来和我配置错误有关系，再最后看到有人花了 12 小时才完成所有题目，瞬间感觉难度还是有的。那这对于未来办活动就有借鉴意义了：

- 我的题目没问题，计分方式可以优化（已经优化完毕，弄完还是挺开心的）

最近龙虾狂热和去年的 mcp 狂热、大模型一体机狂热一样有点...大家真就不带脑子去思考这些事情吗？

# 20260304

周末阴雨天，家人不舒服基本就在家窝着，感觉家里已经捂出了味道，好在昨天病情基本好转了，我似乎也没被感染，虽然嗓子稍微有点难受（从上周似乎就这样）。

上周看完了《寂静的春天》，心情有一丝丝的抑郁。然后周日又看了相关题材的《黑水》，这个就真心有点致郁，原因是杜邦公司半个世纪前做的事，对现在都还有影响，防水防污的材质现在已经深入到各个角落，从不粘锅到冲锋衣、美妆等，而这些制造环节都会有致癌的物质存在，污染治理不得当大家都会受到影响，现在我们已经是这类材料的第一产国，污染已经发生很多年。这么影响大的事情爆出来也就最近 10 年，真的有点匪夷所思...

今天第二次咨询，感觉没有第一次有成效，总是很多事情是我过于”追求完美“，强大的自驱力导致的。比如我会想很多步去提前做一些事情，这些事可能超过了事物本身的需求很多。

现在的问题是，我能坚持这样多久，我的身体能支撑我坚持多久。咨询师说工作的目的是为了更好的生活，好像我现在只有工作没有生活，让我好好思考这些问题。

这周也看完了第二本书《天下之一》，讲的内容超出了技术本身，也涉及到哲学等”虚无“的领域，打算空了好好整理一下笔记。

周末用 AI 写了个文章，昨天发了出去，阅读量极其的低，我一直也觉得现在 AI 的新闻太多了，有点麻木，以及没有价值。

这周看书的同时也思考了下 AI，感觉想通很多东西：

- AGI 不会很快到来，受限于 LLM 现在的形态
- 通用 Agent 不会很普及，受限于 LLM 的价格。虽然 OpenClaw 很火，但很难想到用它的地方，写 PPT 这类不够精，定时任务这类其使用工作流也能实现，总之替代人不太够，助手不够专，还费钱
- 专门的，高度封装过的会很普及，比如各种 Copilot，现在通义、豆包等助手也属于这类
- 通用 Agent 现在更多的是路线和技术的探索，里面一些理念还是不错的，比如记忆压缩（解决长上下文溢出问题），任务分层（也就是任务拆分，不求一个 Agent 搞定所有事，也不是真的用 Multi-Agent，因为这个又会比较复杂）

这个 Q 又拿到一个奖，还拿了个 Kahoot 第三，感觉我还是有一些财运的（除了股市，这两天因为伊朗的事绿的不行）

CC 的材料还没整理完，上周整理了半周，东西是真的多，不过 95% 的完成率，这周还是要弄完。

总部回来的 AI 材料还没整理完，这周感觉也是要弄完。

我的自驱力又犯病了。

接着是锦绣中华，好像里面是很多民俗相关的东西，有点和世界之窗类似，只是偏向于国内版本，到的时候已经两点多了，老人不舒服最后也没进去，直接回去休息了。晚上商场转了转，感觉对这些没有兴趣的时候挺无聊的，听着音乐，也不知道干啥。（人就是不能太无聊，又或者我是耐不住无聊）

最后一天回家，机场吃了很贵的茶餐厅，挺搞笑，在广东没吃，回到上海吃了...
